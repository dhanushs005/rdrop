Reddrop

A smart MP4 video downloader using yt-dlp.
It installs dependencies, simulates download size, and gets the job done fast.

Install on Debian/Ubuntu

echo "deb [trusted=yes] https://dhanushs005.github.io/rdrop/ ./" | sudo tee /etc/apt/sources.list.d/reddrop.list
sudo apt update
sudo apt install rdrop
